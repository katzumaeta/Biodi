Prototipo

El prototipo cuenta consta de software y hardware. El primero es para mostrar la información del segundo y sea más fácil para el protesista su interpretación.

Muesta del prototipo

En este se aprecia la ausencia de ruido y funcionamiento adecuado en el proceso de medición de presiones.

Mockups del software

El sistema desarrollado cuenta con una primer interfaz de usuario donde se deberá colocar el DNI o código SIS del paciente para poder almacenar los datos en su historial médico. Posteriormente, se mostrarán 4 opciones después del ingreso donde se especifica el nombre, DNI del paciente y demás carácterísticas del paciente a tener en cuenta en el momento de fabricación de socket. También, un menú donde se encontrará una opción para ingreso del sistema de toma de medidas y otro dónde se mostrarán los resultados de dichas mediciones. Finalmente, se tiene una opción de ayuda dónde el protesista puede acceder cuando ocurra un problema con el dispositivo.
